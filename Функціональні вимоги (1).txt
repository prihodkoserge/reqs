    1. Функціональні вимоги
        1.1. Система повинна виконувати розподіл ролей на відвідувача (неавторизований користувач), Адміністратора та Користувача (авторизований в системі відвідувач).
        1.2. Система повинна дозволяти Відвідувачу переглядати записи на сайті.
        1.3. Система повинна дозволяти Відвідувачу сайту реєструватись в системі.
        1.4. Система повинна надавати можливість Відвідувачу авторизовуватись в системі.
        1.5. Система повинна надавати можливість Користувачу переглядати записи на форумі.
        1.6. Система повинна надавати можливість Користувачу переглядати конкретний пост та коментарі до нього.
        1.7. Система повинна дозволяти авторизованому Користувачу залишати коментарі.
        1.8. Система повинна дозволяти Користувачу виконувати пошук по форуму за такими параметрами:
    • Дата публікації;
    • Ключові слова публікації.
        1.9. Система повинна надавати можливість Адміністратору публікувати, редагувати та видаляти пости.
        1.10. Система повинна автоматично додавати дату та час при публікуванні запису Адміністратором.
        1.11. Система повинна надавати можливість Адміністратору встановлювати ключові слова (хештеги) при публікації посту. 
        1.12. Система повинна надавати можливість Адміністратору прикріпляти медіаконтент до посту.
        1.13. Система повинна надавати можливість Адміністратору видаляти коментарі.
        1.14. Система повинна автоматично додавати дату та час при публікуванні запису Адміністратором.
    2. Нефункціональні вимоги
        2.1. Система повинна бути реалізована як плагін для інтеграції з існуючою системою.
        2.2. Завантаження веб-сторінки системи повинно бути не більше ніж 3с.
        2.3. Відповідь від системи на певний запит повинна бути не більше 2с.
        2.4. Система повинна коректно працювати в таких браузерах:
    • Chrome (версія 50.0 та вище)
    • Mozilla Firefox (версія 45.0 та вище)
    • Opera (версія 30.0 та вище)
    • IE (версія 10.0 та вище )
    • Safari (версія 5.0 та вище)
        2.5. Система повинна коректно відображатись на мобільних пристроях з такою операційною системою:
    • Android(версія 4.4 та вище);
    • IOS(версія 7.0 та вище).
